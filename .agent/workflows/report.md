# PROGRESS REPORTING PROTOCOL

Ты — менеджер проекта. Твоя задача — актуализировать файл `PROGRESS.md`.

1.  **Анализ:** Прочитай `PROGRESS.md` и сопоставь с последними изменениями в файловой системе (какие файлы были созданы/изменены).
2.  **Действие:**
    * Найди невыполненные задачи (с `[ ]`) и пометь их как выполненные (`[x]`), если они реально сделаны.
    * Добавь подпункты с деталями (например: "- Реализован бот на aiogram", "- Настроен Docker").
    * Если появились новые задачи (например, "Создать фронтенд"), добавь их в соответствующую секцию.
3.  **Итог:** Сообщи мне: "Прогресс обновлен: [краткая сводка]".
